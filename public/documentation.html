Overview:

This project is a server-side component for a movies web app. The application will give users information about different things related to movies, for example, directors, genres, actors, etc. Users will be able to sign up, update their own personal info, and compile a list of their favorite movies.

Endpoints:

  • The endpoint /users supports:
    • POST: Creates a new user.
    • GET: Returns a list of all users in JSON format.
  • The endpoint /users/:name/movies/:MovieID supports:
    • POST: Adds a movie to a user's list of favorite movies.
    • DELETE: Removes a movie from a user's list of favorite movies.
  • The endpoint /users/:name supports:
    • GET: Retrieves a specific user by their name.
    • PUT: Updates a user's information.
    • DELETE: Removes a user from the database.
  • The endpoint /movies supports:
    • GET: Returns a list of all movies in JSON format.
  • The endpoint /movies/:movieTitle supports:
    • GET: Retrieves a single movie by its title.
  • The endpoint /movies/genre/:genreName supports:
    • GET: Retrieves details of a genre by its name.
  • The endpoint /movies/director/:directorName supports:
    • GET: Retrieves details of a director by their name.
  • The endpoint / supports:
    • GET: Displays a welcome message.
